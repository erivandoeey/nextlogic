# NextLogic

Sistema inteligente de gestão imobiliária desenvolvido com foco em automação, organização e produtividade para imobiliárias e corretores.

---

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

---

## Requisitos

Antes de iniciar, certifique-se de possuir instalado:

- Node.js
- npm

Download oficial:

https://nodejs.org/

---

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

### 2. Acessar a pasta do projeto

```bash
cd nextlogic-main
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Executar o servidor local

```bash
npm run dev
```

---

## Acesso local

O projeto estará disponível em:

```text
http://localhost:8000
```
